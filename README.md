# RaquelSkills

Biblioteca de skills jurídicos e processamento de documentos.

## Estrutura
- **_compartilhados/**: Rotinas, templates, cálculos reutilizáveis
- **skills/**: Skills individuais
- **_backups/**: Backups automáticos (não versionar)
- **examples/**: Casos-teste e exemplos
- **_APAGAR/**: Scratch area temporária

## Requerimentos
- Python 3.10+
- Tesseract 5.x
- Whisper (openai-whisper)
