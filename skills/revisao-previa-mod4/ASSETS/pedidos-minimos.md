# Pedidos mínimos por tipo de ação

Consumido por: revisao-previa-mod4 §3 D2.
Data de referência: 2026-05-12.

Cada pedido marcado como [OBRIGATÓRIO] é bloqueante quando ausente.
Pedidos marcados como [RECOMENDADO] são toleráveis quando ausentes, mas a skill os propõe.
