# Rotina pós-protocolo

Executar imediatamente após confirmação de protocolo bem-sucedido.
Tempo estimado de execução: 10–15 minutos.

---

## Etapa 1 — Registro imediato (fazer antes de fechar o sistema)

| # | Ação | Feito |
|---|---|---|
| 1.1 | Salvar o número do protocolo (token de distribuição ou número provisório) | |
| 1.2 | Registrar data e hora exata do protocolo | |
| 1.3 | Capturar print do recibo de protocolo ou salvar PDF gerado pelo sistema | |
| 1.4 | Registrar o nome do juízo/vara onde foi distribuído (quando distribuição imediata) | |

---

## Etapa 2 — Atualização do dossiê do caso

| # | Ação | Feito |
|---|---|---|
| 2.1 | Inserir número do processo no campo `numero_processo` do dossie-caso | |
| 2.2 | Atualizar `fase_atual` → "inicial_protocolada" | |
| 2.3 | Atualizar `status_atual` e `proxima_acao` | |
| 2.4 | Registrar o juízo distribuído e o julgador (quando disponível) | |

---

## Etapa 3 — Comunicação com o cliente

| # | Ação | Feito |
|---|---|---|
| 3.1 | Enviar confirmação de protocolo ao cliente (WhatsApp ou e-mail) | |
| 3.2 | Informar número do processo e nome do juízo | |
| 3.3 | Explicar próximos passos: prazo de resposta do réu e possível audiência | |
| 3.4 | Orientar cliente a NÃO aceitar propostas extrajudiciais sem consultar o escritório | |

**Modelo de mensagem (WhatsApp):**
> "Bom dia, [Nome]. Sua ação foi protocolada com sucesso hoje, [data].
> Número do processo: [nº]. Juízo: [vara/juizado].
> O prazo para a parte contrária responder é de [15 dias — JEF / 30 dias — vara cível].
> Assim que houver movimentação, comunicaremos imediatamente. Qualquer dúvida, estamos à disposição."

---

## Etapa 4 — Configuração de prazos e alertas

| # | Ação | Feito |
|---|---|---|
| 4.1 | Configurar alerta de prazo de contestação: JEF 15 dias úteis; Vara Cível/Trabalhista 30 dias úteis; INSS 30 dias (art. 188 CPC — prazo em dobro) | |
| 4.2 | Configurar alerta de verificação de movimentação: 7 dias após protocolo | |
| 4.3 | Configurar alerta de despacho inicial: verificar em 10 dias úteis | |
| 4.4 | Para tutela de urgência: verificar decisão em 24h | |

---

## Etapa 5 — Verificação de distribuição

| # | Ação | Feito |
|---|---|---|
| 5.1 | Acessar o sistema em 24–48h para confirmar distribuição e número definitivo do processo | |
| 5.2 | Verificar se houve despacho de emenda à inicial (prazo começa a correr da intimação) | |
| 5.3 | Verificar se o sistema gerou citação automática (PJe em alguns juízos) | |
| 5.4 | Atualizar número do processo definitivo no dossiê | |

---

## Etapa 6 — Para tutela de urgência/liminar

| # | Ação | Feito |
|---|---|---|
| 6.1 | Verificar sistema a cada 4h nas primeiras 24h após protocolo | |
| 6.2 | Se deferida: comunicar cliente imediatamente + providenciar cumprimento | |
| 6.3 | Se indeferida: avaliar recurso (agravo de instrumento — 15 dias, art. 1.003 §5º CPC) | |
| 6.4 | Para benefício previdenciário: liminar pode ser cumprida diretamente pelo INSS via ofício | |

---

## Etapa 7 — Atualização do sistema interno

| # | Ação | Feito |
|---|---|---|
| 7.1 | Registrar o processo no sistema de gestão do escritório (se houver) | |
| 7.2 | Iniciar planilha de acompanhamento processual (ou atualizar existente) | |
| 7.3 | Arquivar fisicamente os documentos originais (se não digitalizados) | |
| 7.4 | Registrar honorários contratuais na planilha financeira | |
