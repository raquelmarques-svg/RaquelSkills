## V1 — Verbos e expressões proibidos em peças processuais

| Proibido | Substituir por |
|---|---|
| "Cumpre ressaltar" | Omitir — começar pelo conteúdo |
| "Importante destacar" | Omitir — começar pelo conteúdo |
| "Conforme será demonstrado" | Omitir — demonstrar diretamente |
| "Em síntese" (no meio da peça) | Omitir ou "Portanto," |
| "Não restam dúvidas" | Omitir — o argumento demonstra por si |
| "É notório que" | Omitir — se é notório, dispensa afirmação |
| "Robusto" / "sólido" | Especificar o que torna a prova forte |
| "Relevante" / "importante" | Especificar a relevância concretamente |
| "Cristalino" / "inconteste" | Omitir — o argumento demonstra |
| "Com todo respeito" | Omitir — objetar diretamente |
| "Data venia" | Omitir em peças correntes (reservar para recursos excepcionais) |
| "Milita" (em vez de "opera") | "Opera", "aplica-se", "incide" |
| "Resta comprovado" | "Está comprovado" / "ficou demonstrado" |
| "Diante do exposto" | "Por essas razões," / "Pelo exposto," |

## V2 — Travessão intercalador — proibido

Jamais usar travessão (—) como marcador de inciso, aposto ou incidental em peças processuais e documentos do escritório.

**Errado:** "O benefício — espécie B91 — foi negado."
**Correto:** "O benefício (espécie B91) foi negado." ou "O benefício de espécie B91 foi negado."

---

## V3 — Estrutura "não X, mas Y" — proibido

**Errado:** "Não se trata de liberalidade, mas de direito líquido e certo."
**Correto:** "O pedido tem amparo em direito líquido e certo, não em mera liberalidade."

## V4 — Jargões jurídicos a evitar por inflação de uso

| Jargão | Substituir por |
|---|---|
| "Hodiernamente" | "Atualmente", "hoje" |
| "Outrossim" (uso excessivo) | "Além disso", "ademais" |
| "Porquanto" (quando causal) | "Porque", "dado que" |
| "Mormente" | "Sobretudo", "especialmente" |
| "Dessarte" | "Assim", "portanto" |
| "Forçoso reconhecer" | Omitir — reconhecer diretamente |
